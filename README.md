# Athlete Readiness Dashboard

A full-stack web app to log daily recovery + training metrics and visualize readiness trends.

## MVP Features
- Daily check-in (sleep, soreness, stress, RPE, minutes)
- Readiness score (0–100)
- 7/30-day trend charts

## Tech Stack
- Frontend: React (Vite)
- Backend: Node.js + Express
- Database: SQLite (dev)
